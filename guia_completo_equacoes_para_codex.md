# Módulo 1 — Expressões algébricas

## Objetivo

Introduzir letras, coeficientes, termos e operações algébricas.

## Conteúdos

### O que é uma expressão algébrica

Explicar que uma expressão algébrica combina:

- Números;
- Letras;
- Operações matemáticas.

Exemplos:

```text
3x + 5
2a - 4b
x² + 2x + 1
```

### Termos de uma expressão

No exemplo:

```text
4x² - 3x + 7
```

Os termos são:

```text
4x²
-3x
7
```

### Coeficiente

Explicar que o coeficiente é o número que multiplica a parte literal.

Exemplos:

```text
5x → coeficiente 5
-3y → coeficiente -3
x → coeficiente 1
-x → coeficiente -1
```

### Parte literal

Explicar que a parte literal é formada pelas letras e seus expoentes.

Exemplos:

```text
7x² → parte literal x²
4ab → parte literal ab
-2x³y → parte literal x³y
```

### Grau de um termo

Exemplos:

```text
5x → grau 1
3x² → grau 2
4x²y → grau 3
```

## Prática

Criar exercícios para:

- Identificar termos;
- Identificar coeficientes;
- Identificar partes literais;
- Identificar o grau dos termos;
- Classificar expressões.

# Módulo 2 — Redução de termos semelhantes

## Objetivo

Ensinar a identificar e combinar termos semelhantes.

## Conceito principal

Termos semelhantes são termos que possuem exatamente a mesma parte literal, com as mesmas letras e os mesmos expoentes.

Exemplos de termos semelhantes:

```text
3x e 8x
2x² e -5x²
4ab e 7ab
3x²y e -9x²y
```

Exemplos de termos que não são semelhantes:

```text
3x e 3x²
2x e 2y
4ab e 4a
x²y e xy²
```

## Como reduzir termos semelhantes

Somar ou subtrair apenas os coeficientes e conservar a parte literal.

Exemplo 1:

```text
3x + 5x
(3 + 5)x
8x
```

Exemplo 2:

```text
7x - 10x
(7 - 10)x
-3x
```

Exemplo 3:

```text
4x + 3 - 2x + 5
4x - 2x + 3 + 5
2x + 8
```

Exemplo 4:

```text
3x² + 5x - 2x² + 7x
3x² - 2x² + 5x + 7x
x² + 12x
```

## Erros comuns

Não permitir operações incorretas como:

```text
3x + 4y = 7xy
```

O correto é manter:

```text
3x + 4y
```

Também não reduzir:

```text
2x + 3x²
```

## Prática

Criar:

- 20 questões básicas;
- 20 questões intermediárias;
- 20 questões com várias letras;
- 20 questões com potências;
- 10 questões de identificação de erros.

# Módulo 3 — Propriedade distributiva

## Objetivo

Ensinar a multiplicar um termo por todos os termos dentro de parênteses.

## Conceito

```text
a(b + c) = ab + ac
```

## Exemplos

```text
3(x + 4)
3x + 12
```

```text
-2(x - 5)
-2x + 10
```

```text
4(2x + 3)
8x + 12
```

```text
-3(2x - 4)
-6x + 12
```

## Distributiva dupla

Exemplo:

```text
(x + 2)(x + 3)
x² + 3x + 2x + 6
x² + 5x + 6
```

## Erros comuns

Mostrar que:

```text
3(x + 2)
```

não é:

```text
3x + 2
```

O correto é:

```text
3x + 6
```

# Módulo 4 — Equações do primeiro grau

## Objetivo

Ensinar a resolver equações com uma incógnita de expoente 1.

## Forma geral

```text
ax + b = 0
```

com:

```text
a ≠ 0
```

## Conceito de igualdade

Explicar a equação como uma balança:

- Tudo que é feito em um lado deve ser feito no outro;
- O objetivo é isolar a incógnita;
- Operações inversas desfazem operações.

## Operações inversas

```text
Adição ↔ Subtração
Multiplicação ↔ Divisão
Potenciação ↔ Radiciação
```

## Equações simples

Exemplo:

```text
x + 5 = 12
x = 12 - 5
x = 7
```

Exemplo:

```text
3x = 18
x = 18 / 3
x = 6
```

Exemplo:

```text
2x + 4 = 14
2x = 10
x = 5
```

## Incógnita nos dois lados

Exemplo:

```text
5x + 2 = 3x + 10
5x - 3x = 10 - 2
2x = 8
x = 4
```

## Equações com parênteses

Exemplo:

```text
3(x + 2) = 15
3x + 6 = 15
3x = 9
x = 3
```

## Equações com frações

Exemplo:

```text
x/3 + 2 = 6
x/3 = 4
x = 12
```

Exemplo com MMC:

```text
x/2 + x/3 = 10
3x/6 + 2x/6 = 10
5x/6 = 10
5x = 60
x = 12
```

## Equação impossível

Exemplo:

```text
2x + 3 = 2x + 8
3 = 8
```

Conclusão:

```text
Sem solução
```

## Identidade

Exemplo:

```text
2(x + 3) = 2x + 6
2x + 6 = 2x + 6
```

Conclusão:

```text
Infinitas soluções
```

## Verificação da resposta

Sempre substituir o valor encontrado na equação original.

Exemplo:

```text
2x + 4 = 14
x = 5
```

Verificação:

```text
2 × 5 + 4 = 14
10 + 4 = 14
14 = 14
```

## Prática

Criar níveis:

### Nível 1

Equações simples:

```text
x + a = b
ax = b
ax + b = c
```

### Nível 2

- Incógnita nos dois lados;
- Números negativos;
- Parênteses;
- Redução de termos semelhantes.

### Nível 3

- Frações;
- Decimais;
- Vários parênteses;
- Equações impossíveis;
- Identidades.

Quantidade mínima:

- 30 questões básicas;
- 30 intermediárias;
- 30 avançadas;
- 20 problemas contextualizados.

# Módulo 5 — Problemas com equações do primeiro grau

## Objetivo

Ensinar a transformar situações escritas em equações.

## Método sugerido

1. Ler o problema;
2. Identificar o que é desconhecido;
3. Escolher uma letra para representar o desconhecido;
4. Traduzir as informações para linguagem matemática;
5. Montar a equação;
6. Resolver;
7. Verificar;
8. Responder com uma frase completa.

## Traduções frequentes

```text
Um número → x
O dobro de um número → 2x
O triplo de um número → 3x
A metade de um número → x/2
Um número aumentado em 5 → x + 5
Um número diminuído em 7 → x - 7
A soma de dois números consecutivos → x + (x + 1)
```

## Exemplo

Problema:

```text
O dobro de um número, somado com 6, é igual a 20.
Qual é esse número?
```

Equação:

```text
2x + 6 = 20
```

Resolução:

```text
2x = 14
x = 7
```

Resposta:

```text
O número é 7.
```

# Módulo 6 — Inequações do primeiro grau

## Objetivo

Ensinar desigualdades com uma incógnita de grau 1.

## Símbolos

```text
>  maior que
<  menor que
≥  maior ou igual a
≤  menor ou igual a
```

## Resolução básica

Exemplo:

```text
3x + 2 > 11
3x > 9
x > 3
```

## Regra fundamental

Ao multiplicar ou dividir os dois lados por um número negativo, inverter o sinal da desigualdade.

Exemplo:

```text
-2x > 8
x < -4
```

## Representação na reta numérica

Explicar:

- Círculo aberto para `>` ou `<`;
- Círculo fechado para `≥` ou `≤`;
- Seta indicando o conjunto de soluções.

## Intervalos

Exemplos:

```text
x > 3 → ]3, +∞[
x ≥ 3 → [3, +∞[
x < 5 → ]-∞, 5[
x ≤ 5 → ]-∞, 5]
```

Pode-se também apresentar a notação:

```text
(3, +∞)
[3, +∞)
(-∞, 5)
(-∞, 5]
```

## Inequações simultâneas

Exemplo:

```text
2 < x + 1 ≤ 6
1 < x ≤ 5
```

# Módulo 7 — Sistemas de equações do primeiro grau

## Objetivo

Resolver conjuntos de equações com duas ou mais incógnitas.

## Sistema 2 × 2

Exemplo:

```text
x + y = 10
x - y = 2
```

Solução:

```text
x = 6
y = 4
```

## Método da substituição

Exemplo:

```text
x + y = 10
x = y + 2
```

Substituir:

```text
y + 2 + y = 10
2y = 8
y = 4
```

Depois:

```text
x = 4 + 2
x = 6
```

## Método da adição ou eliminação

Exemplo:

```text
x + y = 10
x - y = 2
```

Somando:

```text
2x = 12
x = 6
```

Depois:

```text
6 + y = 10
y = 4
```

## Método da comparação

Isolar a mesma incógnita nas duas equações e igualar as expressões.

## Interpretação gráfica

Explicar que cada equação representa uma reta.

### Sistema possível e determinado

As retas se cruzam em um único ponto.

### Sistema possível e indeterminado

As retas coincidem.

### Sistema impossível

As retas são paralelas.

## Sistemas 3 × 3

Introduzir apenas depois do domínio dos sistemas 2 × 2.

Trabalhar com:

- Eliminação;
- Substituição;
- Escalonamento elementar.

## Problemas

Criar problemas envolvendo:

- Preço de produtos;
- Quantidade de ingressos;
- Idades;
- Misturas simples;
- Animais e número de patas;
- Moedas e cédulas;
- Produção;
- Distância e velocidade.

## Prática

Criar:

- 20 sistemas por substituição;
- 20 sistemas por adição;
- 15 sistemas por comparação;
- 15 questões de classificação;
- 20 problemas contextualizados;
- 15 sistemas 3 × 3.

# Módulo 8 — Função do primeiro grau

## Objetivo

Relacionar equações do primeiro grau com gráficos de retas.

## Forma geral

```text
f(x) = ax + b
```

## Elementos

### Coeficiente angular

```text
a
```

Indica a inclinação da reta.

### Coeficiente linear

```text
b
```

Indica onde a reta corta o eixo vertical.

## Crescimento e decrescimento

```text
a > 0 → função crescente
a < 0 → função decrescente
a = 0 → função constante
```

## Raiz da função

Encontrar o valor de `x` para o qual:

```text
f(x) = 0
```

Exemplo:

```text
f(x) = 2x - 6
2x - 6 = 0
2x = 6
x = 3
```

## Construção de gráfico

Criar uma tabela de valores.

Exemplo:

```text
f(x) = 2x + 1
```

Tabela:

```text
x = -1 → y = -1
x = 0  → y = 1
x = 1  → y = 3
```

## Interseção entre retas

Relacionar o ponto de interseção com a solução de um sistema.

## Prática

Criar:

- Identificação de coeficientes;
- Cálculo de imagens;
- Cálculo de raízes;
- Classificação em crescente ou decrescente;
- Construção de tabelas;
- Leitura de gráficos;
- Interseção de retas.

# Módulo 11 — Equações do segundo grau

## Objetivo

Resolver equações com incógnita de grau 2.

## Forma geral

```text
ax² + bx + c = 0
```

com:

```text
a ≠ 0
```

## Equações incompletas

### Caso `ax² + bx = 0`

Exemplo:

```text
x² - 5x = 0
x(x - 5) = 0
x = 0 ou x = 5
```

### Caso `ax² + c = 0`

Exemplo:

```text
x² - 16 = 0
x² = 16
x = 4 ou x = -4
```

## Fórmula de Bhaskara

```text
Δ = b² - 4ac
```

```text
x = (-b ± √Δ) / 2a
```

## Interpretação do discriminante

```text
Δ > 0 → duas raízes reais diferentes
Δ = 0 → uma raiz real dupla
Δ < 0 → nenhuma raiz real
```

## Exemplo completo

```text
x² - 5x + 6 = 0
```

Coeficientes:

```text
a = 1
b = -5
c = 6
```

Discriminante:

```text
Δ = (-5)² - 4 × 1 × 6
Δ = 25 - 24
Δ = 1
```

Raízes:

```text
x = (5 ± 1) / 2
x₁ = 3
x₂ = 2
```

## Relações entre raízes

```text
x₁ + x₂ = -b/a
x₁ × x₂ = c/a
```

## Métodos de resolução

Ensinar:

- Fatoração;
- Bhaskara;
- Completar quadrados;
- Relações entre raízes.

## Prática

Criar:

- 20 equações incompletas;
- 30 equações completas;
- 20 por fatoração;
- 20 com análise de discriminante;
- 20 problemas contextualizados.

4. Escolher os intervalos corretos.

## Exemplo


```text
x < 2 ou x > 3
```

## Inequação produto

Exemplo:

```text
(x - 1)(x + 4) ≤ 0
```

## Inequação quociente

Exemplo:

```text
(x - 2) / (x + 1) > 0
```

Explicar restrições de domínio.

# Módulo 14 — Equações fracionárias

## Objetivo

Resolver equações nas quais a incógnita aparece em denominadores.

## Passos

1. Determinar as restrições;
2. Encontrar o MMC algébrico;
3. Multiplicar todos os termos pelo MMC;
4. Resolver a equação resultante;
5. Verificar se as soluções respeitam as restrições.

## Exemplo

```text
2/(x - 1) = 3
```

Restrição:

```text
x ≠ 1
```

Resolução:

```text
2 = 3(x - 1)
2 = 3x - 3
3x = 5
x = 5/3
```

## Prática

Criar exercícios com:

- Um denominador algébrico;
- Vários denominadores;
- Fatoração nos denominadores;
- Soluções inválidas;
- Problemas contextualizados.

# Módulo 15 — Equações irracionais

## Objetivo

Resolver equações com incógnitas dentro de raízes.

## Passos

1. Determinar as condições de existência;
2. Isolar a raiz;
3. Elevar os dois lados à potência adequada;
4. Resolver;
5. Verificar todas as respostas na equação original.

## Exemplo

```text
√(x + 3) = x - 1
```

Condições:

```text
x + 3 ≥ 0
x - 1 ≥ 0
```

Depois elevar ao quadrado:

```text
x + 3 = (x - 1)²
```

Explicar que podem surgir soluções estranhas ou falsas.

## Prática

Criar:

- Equações com uma raiz;
- Equações com duas raízes;
- Equações que geram soluções falsas;

# Módulo 16 — Equações e inequações modulares

## Objetivo

Interpretar módulo como distância.

## Conceito

```text
|x| = distância de x até 0
```

## Equação modular

```text
|x - 3| = 5
```

Casos:

```text
x - 3 = 5
```

ou:

```text
x - 3 = -5
```

Soluções:

```text
x = 8
x = -2
```

## Inequações modulares

Exemplo:

```text
|x| < 4
```

Equivale a:

```text
-4 < x < 4
```

Exemplo:

```text
|x| > 4
```

Equivale a:

```text
x < -4 ou x > 4
```

## Prática

Criar:

- Equações modulares simples;
- Equações modulares com deslocamento;
- Inequações modulares;

- Problemas de distância.

# Módulo 17 — Potenciação e equações exponenciais

## Objetivo

Resolver equações em que a incógnita aparece no expoente.

## Propriedades das potências

```text
a^m × a^n = a^(m+n)
a^m / a^n = a^(m-n)
(a^m)^n = a^(mn)
(ab)^n = a^n b^n
a^0 = 1
a^(-n) = 1/a^n
```

## Igualação de bases

Exemplo:

```text
2^(x + 1) = 16
```

Como:

```text
16 = 2^4
```

Então:

```text
x + 1 = 4
x = 3
```

## Substituição

Exemplo:

```text
4^x - 5·2^x + 4 = 0
```

Fazer:

```text
y = 2^x
```

Então:

```text
y² - 5y + 4 = 0
```

## Prática

Criar:

- Igualação de bases;
- Uso de propriedades;
- Substituição;
- Equações com bases diferentes;
- Problemas de crescimento e decaimento.

# Módulo 18 — Logaritmos e equações logarítmicas

## Objetivo

Introduzir logaritmos e resolver equações logarítmicas.

## Definição

```text
log_a(b) = x
```

equivale a:

```text
a^x = b
```

## Condições

```text
a > 0
a ≠ 1
b > 0
```

## Propriedades

```text
log_a(MN) = log_a(M) + log_a(N)
log_a(M/N) = log_a(M) - log_a(N)
log_a(M^k) = k·log_a(M)
```

## Exemplo

```text
log₂(x) = 5
```

Equivale a:

```text
x = 2^5
x = 32
```

## Equação logarítmica

Exemplo:

```text
log₂(x - 1) = 3
```

```text
x - 1 = 8
x = 9
```

Verificar domínio:

```text
x - 1 > 0
```

## Prática

Criar:

- Conversão entre forma logarítmica e exponencial;
- Cálculo de logaritmos simples;
- Uso de propriedades;
- Equações logarítmicas;
- Questões de domínio;
- Problemas contextualizados.
